# Task Management App

A production-level MERN Stack Task Management Application built while learning React and Node.js.

## 🚀 Tech Stack

### Frontend
- React.js
- React Router DOM
- Tailwind CSS
- Axios

### Backend
- Node.js
- Express.js
- MongoDB
- JWT Authentication
- Bcrypt

## ✨ Features

- User Authentication
- Register & Login
- Form Validation
- Protected Routes
- CRUD Operations
- Responsive UI
